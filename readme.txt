O sistema possui 6 páginas além da principal:
- Veterinário
- Tosa e banho
- Educação
- Alimentação
- Day care
- Transporte

Cada página representa um serviço/produto do pet shop. Todas contém as funções de navegação entre as páginas do sistema e formulários funcionais para
que o cliente possa marcar uma data e horário para o serviço.
No canto superior direito há um botão que exibirá na tela um formulário de cadastro para o cliente e para seu pet.
No canto superior esquerdo, na logo do pet shop há um botão que permite o usuário voltar para a página principal.
